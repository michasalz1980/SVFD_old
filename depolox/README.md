# Aufgaben zur Verbesserung des Dashboards

## 1. CSS in eigene Datei auslagern
- Trenne alle CSS-Styles vom HTML.
- Erstelle eine separate `.css`-Datei (z.B. `styles.css`).
- Binde die CSS-Datei im `<head>` der HTML-Datei ein.

## 2. JavaScript in eigene Datei auslagern
- Entferne sämtlichen JavaScript-Code aus der HTML-Datei.
- Erstelle eine separate `.js`-Datei (z.B. `main.js`).
- Binde die JS-Datei am Ende des `<body>`-Tags ein.

## 3. Unnötigen Ballast aus API und index entfernen
- Analysiere die `API`- und `index`-Dateien.
- Entferne nicht verwendete oder redundante Funktionen, Variablen und Imports.
- Dokumentiere größere Löschungen oder Umstrukturierungen kurz in den Commits.
- Teste nach dem Entfernen die Grundfunktionen auf Fehlerfreiheit.

---

> **Hinweis:** Nach Abschluss dieser Aufgaben sollte das Dashboard übersichtlicher, modularer und wartbarer sein.
